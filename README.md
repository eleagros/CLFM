Reconstruction of big polarimetric images from a series of tiles.

Install the MATLAB Engine API for Python
Follow the steps present in https://ch.mathworks.com/help/matlab/matlab_external/install-the-matlab-engine-for-python.html, it requires the Image Processing Toolbox in MATLAB.
